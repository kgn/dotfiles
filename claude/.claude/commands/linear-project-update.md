# Generate Linear Project Update

Generate a project update summary based on work completed in this conversation that can be copied into Linear's project update field.

## Instructions

1. Review the conversation history to identify what was accomplished
2. Summarize the work completed, issues resolved, code written, or features implemented
3. Note any work still in progress or blocked
4. Identify next steps
5. Recommend a health status (On track, At risk, Off track)
6. Format for easy copy/paste into Linear's project update field

## Output Format

```
**Health Status Recommendation**: [On track | At risk | Off track]

---

[2-3 sentence summary of what was accomplished]

**Completed:**
- [List of completed items from this session]

**In Progress:**
- [Any incomplete work, or "None" if all complete]

**Next Steps:**
- [What should be done next]

**Blockers/Risks:**
- [Any blockers or risks encountered, or "None"]
```
